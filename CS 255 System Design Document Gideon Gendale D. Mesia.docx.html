Technical Requirements
[Based on the diagrams you have created, describe the technical requirements of your system. These
requirements should address the required hardware, software, tools, and infrastructure necessary for
your system design.]
The requirements for this program would be a cloud-based server, as this is a small business so
it’s much more cost effective in the long run. Because this business has a chance to grow, a cloud-based
server would help it with scalability once the company increases in manpower and size. The hardware
required would preferably be laptops for admins and IT as it is portable and can be moved anywhere.
Although the system doesn’t last as long as computers, the mobility and ability to access the
information anywhere would help the company potentially see out home services and may not even
require an office building to work from. Software would include windows as the operating system, as it
has a multitude of support and used wisely. Something like MacOS might need to come with the
hardware already and may be marked up in price, while Linux might not be user friendly for everyone.
Windows has the support and general ease of use for everyone to be able to use. Though along with that
an office space might be needed as it a cloud-based server, internet is required. Admin and employees
may sometimes not have the best internet connection, so a place with reliable internet is required.
Although we do have IT personnel that work within the company, there should be system software that
eases both the IT and admin personnel, namely daily backups for system and information. Lastly the
DriverPass website needs to be accessible to the consumer with web browsers. An app is not necessarily
needed, but having the website work in windows and MacOS is essential to garter a wide range of
customers/users, while also being able to open not only through computers, but mobile devices as well.
